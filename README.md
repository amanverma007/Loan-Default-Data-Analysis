# Loan-Default-Data-Analysis
# 💳 Loan Default Analysis Dashboard

## 📊 About this project
I worked on this project to understand how loan data behaves and what factors might be linked to loan defaults.

Instead of just looking at numbers, I tried to explore patterns like how income, employment type, age, and credit score impact loan amounts and default rates.

---

## 🛠️ Tools I used
- Power BI  
- Excel  
- DAX (basic calculations)

---

## 📌 What this dashboard includes
- Loan amount distribution based on purpose  
- Average income across different employment types  
- Default rate comparison (by employment type and year)  
- Loan trends across age groups  
- Credit score vs loan amount analysis  
- Some basic demographic breakdown (age, education, etc.)

I also tried to keep the layout simple so it's easy to understand at a glance.

---

## 📈 A few things I noticed
- Default rate is higher for unemployed applicants  
- Full-time employees have higher income but relatively lower default  
- Loan amount generally decreases when credit score risk increases  
- Middle-aged group seems to take more loans overall  
- Education level also shows some variation in loan distribution
